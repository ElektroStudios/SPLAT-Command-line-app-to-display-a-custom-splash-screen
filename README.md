## 👋 Introduction

**SPLAT** is a lightweight command-line utility for Windows designed to display custom splash screens. It allows you to present an image for a specific duration, making it a perfect addition to installation scripts, self-extracting archives (WinRAR, 7z), or batch automation tasks to provide a professional visual layer to your processes.

## 💡 Motivation

When automating software deployments or heavy file extractions, the execution process can often feel "empty" or unresponsive while the background work happens. Adding a splash screen bridges the gap between initiating a process and seeing it finish. It provides instant visual feedback, reinforces branding, and ensures the user knows the application is active and working.

##### ⚡ The Real Question
###### How many times have you launched a script or installer and been left staring at a blank screen, uncertain if the process started or if the system simply froze?.

### ⚙️ Usage

#### Syntax:
```cmd
Splat.exe <IMAGE FILEPATH> [SWITCHES]
```

#### Switches

| Switch | Format | Description | Default |
| :--- | :--- | :--- | :--- |
| **/Duration** | `(ms)` | The splash screen duration. | 5000 ms |
| **/FXDuration** | `(ms)` | The visual effects duration. | 1500 ms |
| **/FadeIN** | *None* | Enable FadeIN effect. | Disabled |
| **/FadeOUT** | *None* | Enable FadeOUT effect. | Disabled |
| **/Resize** | `(widthXheight)` | Set a new size for the image (e.g., 400x400). | Original size |
| **/Location** | `(X,Y)` | Set the position of the image on the current screen (e.g., 100,300). | Center |
| **/Clickable** | *None* | Enables doing click on the image to close it. | Disabled |
| **/OnTop** | *None* | Set the image on top of other windows. | Disabled |
| **/?** | *None* | Shows this help. | — |

---

#### Real World Examples

| Command | Description |
| :--- | :--- |
| `Splat.exe "C:\Image.png"` | Shows an image at the center of the screen for 5000 ms. |
| `Splat.exe "C:\Image.png" /Resize 400x400 /Location 100,300` | Shows a resized image (400x400 px) at 100,300 coordinates for 5000 ms. |
| `Splat.exe "C:\Image.png" /Duration 6000 /FadeIN /FadeOUT /FXDuration 2000` | Shows an image at the center of the screen with fade effects for 6000 ms, plus 2000 ms for each fade effect (Total: 10000 ms). |
